## 🎯 What is Cricfy?

Cricfy is a free APK that lets you stream **live sports** directly on your Android TV or Android device — no subscription, no account, no bullshit.

- ⚽ Football
- 🏏 Cricket
- 🏀 Basketball
- 🎾 Tennis
- 🏆 And more...

**Visit [cricfy.net](https://cricfy.net) for more info.**

---

## 📥 Installation

### Android TV / Google TV

1. Install a file manager app (e.g. **FX File Explorer** or **X-plore**)
2. Enable **Unknown sources** in Settings → Security
3. Download the APK from the [Releases](../../releases/latest) section
4. Open the APK via your file manager and install

### Android Phone / Tablet

1. Download the APK from [Releases](../../releases/latest)
2. Tap the file — allow installation from unknown sources if prompted
3. Install and open
